📡 INTEGRASI GPS TRACKER

1
Pasang GPS tracker Teltonika FMB920 di tiap unit truk. SIM card data aktif (Telkomsel/XL).
2
Gunakan API endpoint untuk update posisi otomatis ke server backend.
3
Panggil PATCH /api/shipments/:id/position setiap 30 detik.
// Contoh update otomatis dari GPS:
fetch(`${API_URL}/api/shipments/TJ-2024-001/position`, {
  method: 'PATCH',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ lat, lng, lokasi, progress })
});
